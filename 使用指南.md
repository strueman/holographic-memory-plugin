## 概述

**Holographic Memory Plugin** 是 Hermes Agent 的外部记忆插件，提供以下核心能力：

- **结构化存储**：按类别、标签、实体组织记忆
- **可信度评分**：记忆有信任分数，越用越准
- **中文支持**：jieba 分词 + LIKE 查询，完美支持中文搜索
- **关联推理**：实体关系、多实体推理、冲突检测

### 与内置 memory 工具的区别

| 特性 | 内置 memory | Holographic fact_store |
|-----|-------------|------------------------|
| 容量限制 | ~2000 字符 | 无限制（SQLite 数据库）|
| 持久化 | 注入到每次会话 | 独立数据库文件 |
| 搜索能力 | 无搜索功能 | 关键词搜索、实体探测、关联推理 |
| 中文支持 | 原生支持 | 需要改造（已完成）|
| 适用场景 | 身份信息、常驻提示 | 长期记忆、结构化知识 |

## 工具清单

Holographic 提供 2 个工具：

| 工具名 | 功能 |
|-------|------|
| `fact_store` | 核心记忆操作（存储、搜索、推理）|
| `fact_feedback` | 对记忆评分，训练信任分数 |

## 快速上手

### 1. 存储记忆

```python
# 基础存储
fact_store(action="add", content="龙哥喜欢喝拿铁，不喜欢太甜的")

# 指定类别
fact_store(action="add", content="本项目使用 Python 3.11", category="project")

# 添加标签
fact_store(action="add", content="WSL 环境下代理端口 7890", tags="devops,network")
```

**类别选项**：`user_pref`（用户偏好）、`project`（项目相关）、`tool`（工具配置）、`general`（通用）

### 2. 搜索记忆

```python
# 关键词搜索
fact_store(action="search", query="龙哥喜欢")

# 限制结果数量
fact_store(action="search", query="Python", limit=5)
```

### 3. 查看所有记忆

```python
fact_store(action="list")
```

### 4. 反馈训练

当你使用某条记忆后发现它有用或过时，可以评分：

```python
# 标记有用（提升信任分数）
fact_feedback(action="helpful", fact_id=1)

# 标记过时（降低信任分数）
fact_feedback(action="unhelpful", fact_id=2)
```

## 功能详解

### action: add — 存储记忆

**参数**：

| 参数 | 必填 | 说明 |
|-----|------|------|
| `content` | ✅ | 记忆内容 |
| `category` | ❌ | 类别：user_pref/project/tool/general |
| `tags` | ❌ | 逗号分隔的标签 |
| `entity` | ❌ | 关联实体（自动提取） |

**示例**：

```python
fact_store(
    action="add",
    content="龙哥的开发环境是 WSL Ubuntu，用户名 ykl",
    category="user_pref",
    tags="environment,wsl"
)
```

**返回**：新建记忆的 ID 和初始信任分数

---

### action: search — 关键词搜索

**参数**：

| 参数 | 必填 | 说明 |
|-----|------|------|
| `query` | ✅ | 搜索关键词 |
| `limit` | ❌ | 最大结果数（默认 10）|
| `min_trust` | ❌ | 最低可信度过滤（默认 0.3）|

**示例**：

```python
fact_store(action="search", query="开发环境", limit=5)
```

**返回**：匹配的记忆列表，按相关性和信任分数排序

---

### action: probe — 实体探测

查询与某个实体相关的所有记忆。

**参数**：

| 参数 | 必填 | 说明 |
|-----|------|------|
| `entity` | ✅ | 实体名称 |
| `limit` | ❌ | 最大结果数 |

**示例**：

```python
fact_store(action="probe", entity="龙哥")
```

**返回**：所有提到"龙哥"的记忆

---

### action: related — 关联实体

查找与某个实体有连接的其他实体。

**参数**：

| 参数 | 必填 | 说明 |
|-----|------|------|
| `entity` | ✅ | 实体名称 |

**示例**：

```python
fact_store(action="related", entity="龙哥")
```

**返回**：与"龙哥"共同出现在记忆中的其他实体

---

### action: reason — 多实体推理

查找同时关联多个实体的记忆。

**参数**：

| 参数 | 必填 | 说明 |
|-----|------|------|
| `entities` | ✅ | 实体名称数组 |
| `limit` | ❌ | 最大结果数 |

**示例**：

```python
fact_store(action="reason", entities=["Python", "项目"])
```

**返回**：同时涉及 Python 和项目的记忆

---

### action: contradict — 冲突检测

查找可能相互矛盾的记忆。

**参数**：

| 参数 | 必填 | 说明 |
|-----|------|------|
| `query` | ❌ | 可选，限定查询范围 |

**示例**：

```python
fact_store(action="contradict")
```

**返回**：可能存在冲突的记忆对

---

### action: update — 更新记忆

**参数**：

| 参数 | 必填 | 说明 |
|-----|------|------|
| `fact_id` | ✅ | 记忆 ID |
| `content` | ❌ | 新内容 |
| `trust_delta` | ❌ | 信任分数调整值 |

**示例**：

```python
fact_store(action="update", fact_id=1, content="更新后的内容")
```

---

### action: remove — 删除记忆

**参数**：

| 参数 | 必填 | 说明 |
|-----|------|------|
| `fact_id` | ✅ | 记忆 ID |

**示例**：

```python
fact_store(action="remove", fact_id=1)
```

---

### action: list — 列出所有记忆

**参数**：

| 参数 | 必填 | 说明 |
|-----|------|------|
| `limit` | ❌ | 最大结果数 |
| `min_trust` | ❌ | 最低可信度过滤 |

**示例**：

```python
fact_store(action="list", limit=20)
```

## 中文搜索说明

### 中文增强改造

本插件已完成中文支持改造，主要改动：

1. **jieba 分词**：中文查询自动分词
2. **LIKE 查询 fallback**：绕过 FTS5 unicode61 的单字分词限制

### 中文搜索能力

| 场景 | 示例 | 支持 |
|-----|------|------|
| 中文关键词 | "龙哥" | ✅ |
| 中文短语 | "开发环境" | ✅ |
| 中英混合 | "Python 项目" | ✅ |
| 单字匹配 | "龙" | ✅ |
| 纯英文专有名词 | "GPT-4" | ⚠️ 部分支持 |

### 已知限制

- **纯英文专有名词**：如 `GPT-4`、`SQLite` 等，LIKE 查询可能无法完美匹配中英文混合内容
- **新插件预热期**：`probe`、`reason` 功能需要积累实体数据，初期可能返回空结果

## 配置选项

编辑 `~/.hermes/config.yaml`：

```yaml
plugins:
  hermes-memory-store:
    # 数据库路径（默认 ~/.hermes/memory_store.db）
    db_path: $HERMES_HOME/memory_store.db
    
    # 是否自动提取实体（默认 false）
    auto_extract: false
    
    # 默认信任分数（默认 0.5，范围 0-1）
    default_trust: 0.5
    
    # 信任阈值（低于此值的记忆不会返回，默认 0.3）
    min_trust_threshold: 0.3
    
    # 时间衰减半衰期（0 表示不衰减）
    temporal_decay_half_life: 0
```

## 常见问题

### Q1: 搜索中文返回空结果？

**检查**：
1. 确认内容已添加：`fact_store(action="list")`
2. 尝试单字搜索测试："龙"
3. 查看数据库：`sqlite3 ~/.hermes/memory_store.db "SELECT * FROM facts;"`

### Q2: 如何备份数据库？

```bash
cp ~/.hermes/memory_store.db ~/.hermes/memory_store.db.bak
```

### Q3: 如何清空所有记忆？

```bash
sqlite3 ~/.hermes/memory_store.db "DELETE FROM facts; DELETE FROM entities;"
```

### Q4: probe/related 返回空？

**原因**：实体表需要随使用积累数据  
**解决**：正常使用一段时间后自动改善

### Q5: 如何切换其他记忆插件？

编辑 `~/.hermes/config.yaml`，修改 `memory_provider` 配置。注意：同一时间只能激活一个外部记忆 provider。

## 技术原理

### 架构

```
fact_store Tool
      ↓
┌─────────────────────────────────┐
│         __init__.py             │
│    (MemoryProvider 接口实现)     │
├─────────────────────────────────┤
│  store.py       │  retrieval.py │
│  (存储层)        │  (检索层)      │
├─────────────────┴───────────────┤
│       SQLite + FTS5             │
│   ~/.hermes/memory_store.db     │
└─────────────────────────────────┘
```

### 中文搜索实现

```
用户查询："龙哥喜欢"
      ↓
┌─────────────────────────────┐
│   _tokenize() 检测中文       │
│   → 使用 jieba 分词          │
│   → ["龙哥", "喜欢"]         │
├─────────────────────────────┤
│   _fts_candidates() 检测中文  │
│   → 切换到 LIKE 查询模式      │
│   → WHERE content LIKE '%龙哥%' │
└─────────────────────────────┘
      ↓
返回匹配结果
```
